## Ingredients
* Computer
* Python
* Atom Text Editor

## Steps

1. Switch your computer on.
    + Keep the terminal handy.

    And a nice wallpaper!

    !["An Image"](http://unsplash.it/500/500?random)
    + Then in Terminal, type:
    ```sh
    atom script_name.py
    ```
2. Write your Python program in Atom.
3. Run your Python program!
